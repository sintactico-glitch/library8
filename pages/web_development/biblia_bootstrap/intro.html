Introducción

En Bootstrap, la concepción inicial del diseño nace de un espacio reducido. Utilizan la filosofía Mobile First: cuando creamos un sitio web en un espacio reducido nos vemos obligados a concentrarnos en el contenido principal del sitio.

Algunos materiales adicionales para entender Mobile First son Mobile First de Luke Wroblewski, The Mobile Book de Smashing Magazine, Mobile First Design with HTML5 and CSS3 de Jason Gonzales.

  • Al armar la estructura del proyecto es recomendable usar los nombres de carpeta css, images y js ya que es la estructura que usan Wordpress, Drupal, etc. Nos ahorra trabajo si luego queremos trabajar con alguno de estos sistemas
  • Para agregar el JavaScript se coloca primero Jquery, después Popper y después Bootstrap JS
  • Es conveniente usar la versión completa de Jquery. La versión slim puede causar problemas con CMS

No es necesario memorizar las clases de Bootstrap, es más eficiente utilizar una cheatsheet para consultar cuando sea necesario.
