Capítulo 2: Colores y tipografía

Entendiendo las medidas REM

Podemos aplicar estilos de forma responsive con los 5 contextos que utiliza Bootstrap.

REM: Medida relativa basada en el tamaño de fuente que tenga la etiqueta HTML.

1rem es el tamaño de texto que se define en la etiqueta HTML. Por defecto suele ser 16px.

La gran ventaja de usar REM es que permite escalar el diseño de forma responsive. Empezamos a trabajar pensando en las versiones mobile, por lo que podemos definir un font-size de 14px y luego ir aumentando el tamaño para contextos mayores. Esto permite tener tipografía escalable dependiendo del tamaño del dispositivo. Podemos usar REM para otros elementos (padding, margin, etc) y también serán responsive.

Se recomienda que el tamaño de fuente para tamaños mobile sea más pequeño.

Clases para encabezados

Con las clases de encabezado de Bootstrap podemos hacer que párrafos tomen el estilo de titulares, sin que sea necesario que sean títulos.

Estas clases toman el valor de .h1 a .h6.

Clases para títulos destacados

En algunas secciones puede ser que necesitemos tamaños mayores para títulos de secciones destacadas. Para esto utilizamos la clase .display.

Los valores son .display-1 al .display-4.

Clases para alineaciones de texto

  • .text-left
  • .text-center
  • .text-right
  • .text-justify

También nos permite alinear el contenido en base al contexto (por ejemplo, .text-sm-left, .text-lg-center).

Clases para transformaciones de texto

  • .text-uppercase
  • .text-lowercase
  • .text-capitalize
  • .font-weight-bold
  • .font-italic
  • .small (80% del tamaño original)

Trabajando con colores

Para empezar a trabajar en el diseño del sitio usamos un color principal y un color secundario. Podemos tomar estos colores del logo del cliente.

Boostrap maneja 6 colores principales:

  • Primary (.text-primary)
  • Secondary (.text-secondary)
  • Success (.text-success)
  • Danger (.text-danger)
  • Warning (.text-warning)
  • Info (.text-info)

También tiene las clases:

  • .text-light y .text-white (para usar con un background oscuro)
  • .text-dark y .text-black (para usar con un background claro)

Las clases que podemos usar para cambiar el fondo de los elementos son:

  • .bg-primary
  • .bg-secondary
  • .bg-warning
  • Toman todos los valores vistos para los colores

Cuando empezamos a trabajar en nuestro sitio vamos a tener que definir los colores primarios y secundarios en base a nuestro cliente. Por lo que tenemos que definir estos colores con los valores personalizados.

Para hacerlo vamos a rescribir las clases de Bootstrap en nuestro css. Usamos el mismo nombre de clase y lo guardamos en el css personalizado.
